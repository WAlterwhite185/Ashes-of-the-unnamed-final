ASHES OF THE UNNAMED — FULL (CDN)
==================================
What’s inside
-------------
• Full vertical slice using Three.js from CDN (works on GitHub Pages & Google Sites)
• Ornate HUD, animated bars, scanlines, vignette
• Melee/parry, Eldritch Bolt (E), Grave Lance (Q), shrines (3), enemy wave, boss with 2 phases
• Pointer-lock after click; audio gated by start button

How to host
-----------
1) Upload 'index.html' to your GitHub repo (root or /docs depending on Pages settings).
2) Enable GitHub Pages for that folder.
3) In Google Sites → Insert → Embed → paste your Pages URL.
4) Resize to 16:9 and click “Start”, then click the game to capture mouse.

Tip: If you later want fully local modules, download:
  - build/three.module.js  → save as /libs/three.module.js
  - examples/jsm/controls/PointerLockControls.js → /libs/PointerLockControls.js
…and update the import paths accordingly.
